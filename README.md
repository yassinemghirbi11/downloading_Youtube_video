# YouTube Video Downloader

A simple Python script to download YouTube videos in MP4 format. This project uses **Pytube** to fetch and download videos.

---

## Features

- Download YouTube videos in MP4 format.
- Automatically selects the highest resolution available.
- Handles both standard and shortened YouTube URLs.
- Saves videos to a `downloads` folder.

---

## Requirements

- Python 3.8 or higher
- [Pytube](https://pypi.org/project/pytube/)

Install Pytube using pip:

```bash
pip install --upgrade pytube
